# Lifetrek Mirror

This is the mirrored repository for the Lifetrek Medical application.

## Project Info

**URL**: https://lifetrek-medical.com

## Setup & Development

### Prerequisites
- Node.js & npm

### Installation

```sh
# Install dependencies
npm install
```

### Development

```sh
# Start development server
npm run dev
```

### Build

```sh
# Build for production
npm run build
```

## Technologies
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
