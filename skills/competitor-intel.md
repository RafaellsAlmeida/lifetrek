# Competitor Intelligence Skill

## Research Framework

### 1. Identify Competitors
**Direct Competitors** (Same services, same region):
- Search: "usinagem dispositivos médicos Brasil"
- Check ANVISA registry for similar manufacturers

**Indirect Competitors** (Adjacent services):
- Contract manufacturers
- Equipment suppliers
- International players entering Brazil

### 2. Data Collection

| Dimension | Sources |
|-----------|---------|
| Company Info | LinkedIn, CNPJ lookup, website |
| Capabilities | Product pages, case studies |
| Certifications | ANVISA, ISO registry, FDA FURLS |
| Pricing | RFQ if possible, industry benchmarks |
| Marketing | LinkedIn posts, Google Ads, SEO rankings |
| Team | LinkedIn, leadership profiles |

### 3. Analysis Template

```markdown
## [Competitor Name]

**Overview**: [1-2 sentence summary]

**Strengths**:
- [Capability 1]
- [Certification]

**Weaknesses**:
- [Gap 1]
- [Limitation]

**Key Differentiator**: [What makes them unique]

**Threat Level**: [Low/Medium/High]

**Opportunity**: [How Lifetrek can differentiate]
```

### 4. Competitive Matrix

| Factor | Lifetrek | Comp A | Comp B |
|--------|----------|--------|--------|
| ISO 13485 | ✓ | ✓ | ? |
| 5-Axis CNC | ✓ | ✓ | ✗ |

## Web Research Tools
- Use `search_web` for current information
- Check LinkedIn company pages
- Review Google Maps reviews
- Analyze their SEO keywords

## Output Format
Deliver as:
1. Executive summary (1 paragraph)
2. Competitor profiles (1 per competitor)
3. Competitive matrix
4. Strategic recommendations
